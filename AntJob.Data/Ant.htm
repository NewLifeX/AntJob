作业（Job）

名称 显示名 类型 长度 精度 主键 允许空 备注
ID 编号 Int32 AI N
AppID 应用 Int32 N
Name 名称 String 100
ClassName 类名 String 100 支持该作业的处理器实现
DisplayName 显示名 String 50
Enable 启用 Boolean N
Mode 调度模式 Int32 N 定时调度只要达到时间片开头就可以跑，数据调度要求达到时间片末尾才可以跑
Cron 执行频次 String 50 定时调度的Cron表达式
Topic 主题 String 50 消息调度时消费的主题
MessageCount 消息数 Int32 N
DataTime 数据时间 DateTime 下一次处理数据的时间，默认从当前时间开始
End 结束 DateTime 到该时间停止调度作业，默认不设置时永不停止
Step 步进 Int32 N 切分任务的时间区间，秒
BatchSize 批大小 Int32 N 数据调度每次抽取数据的分页大小，或消息调度每次处理的消息数，定时调度不适用
Offset 偏移 Int32 N 距离AntServer当前时间的秒数，避免因服务器之间的时间误差而错过部分数据，秒
MaxTask 并行度 Int32 N 一共允许多少个任务并行处理，多执行端时平均分配，确保该作业整体并行度
MaxError 最大错误 Int32 N 连续错误达到最大错误数时停止
MaxRetry 最大重试 Int32 N 默认10次，超过该次数后将不再重试
MaxTime 最大执行时间 Int32 N 默认600秒，超过该时间则认为执行器故障，将会把该任务分配给其它执行器
MaxRetain 保留 Int32 N 任务项保留天数，超过天数的任务项将被删除，默认30天
MaxIdle 最大空闲时间 Int32 N 默认3600秒，超过该时间不更新则认为应用程序故障，系统触发告警
ErrorDelay 错误延迟 Int32 N 默认60秒，出错延迟后重新发放
Deadline 最后期限 DateTime 超过该时间后，任务将不再执行
Total 总数 Int64 N 任务处理的总数据，例如数据调度抽取得到的总行数，定时调度默认1
Success 成功 Int64 N 成功处理的数据，取自于Handler.Execute返回值，或者ProcessItem返回true的个数
Error 错误 Int32 N
Times 次数 Int32 N
Speed 速度 Int32 N
LastStatus 最后状态 Int32 N 最后一次状态
LastTime 最后时间 DateTime 最后一次时间
Data 数据 String Sql模板或C#模板
CreateUserID 创建人 Int32 N
CreateUser 创建者 String 50
CreateTime 创建时间 DateTime
CreateIP 创建地址 String 50
UpdateUserID 更新人 Int32 N
UpdateUser 更新者 String 50
UpdateTime 更新时间 DateTime
UpdateIP 更新地址 String 50
Remark 备注 String 2000


作业任务（JobTask）

名称 显示名 类型 长度 精度 主键 允许空 备注
ID 编号 Int32 AI N
AppID 应用 Int32 N
JobID 作业 Int32 N
Client 客户端 String 50 IP加进程
DataTime 数据时间 DateTime 大于等于，定时调度到达该时间点后触发（可能有偏移量），消息调度不适用
End 结束 DateTime 小于不等于，数据调度到达该时间点后触发（可能有偏移量），消息调度不适用
BatchSize 批大小 Int32 N 数据调度每次抽取数据的分页大小，或消息调度每次处理的消息数，定时调度不适用
Total 总数 Int32 N 任务处理的总数据，例如数据调度抽取得到的总行数，定时调度默认1
Success 成功 Int32 N 成功处理的数据，取自于Handler.Execute返回值，或者ProcessItem返回true的个数
Error 错误 Int32 N
Times 次数 Int32 N
Speed 速度 Int32 N 每秒处理数，执行端计算
Cost 耗时 Int32 N 秒，执行端计算的执行时间
FullCost 全部耗时 Int32 N 秒，从任务发放到执行完成的时间
Status 状态 Int32 N
MsgCount 消息 Int32 N 消费消息数
Server 服务器 String 50
ProcessID 进程 Int32 N
Key 最后键 String 50 Handler内记录作为样本的数据
Data 数据 String 可以是Json数据，比如StatID
Message 消息内容 String Handler内记录的异常信息或其它任务消息
TraceId 追踪 String 200 链路追踪，用于APM性能追踪定位，还原该事件的调用链
CreateIP 创建地址 String 50
CreateTime 创建时间 DateTime
UpdateIP 更新地址 String 50
UpdateTime 更新时间 DateTime
